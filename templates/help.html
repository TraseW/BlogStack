How to use the admin panel.

The panel is pretty simple to use. The main parts are the naming area, the content, the meta, and the submit/delete buttons.

The name

The name input is how you decide on the page that you are editing, submitting, or deleting. Anytime you want to make a change, the backend of the site will make the change to whatever file is specified in the name field.

The content

The content area allows you to create new tags for the page with arbitrary tags to use. So you can use nearly any tag. There is no nesting of tags in this system though, so anything nested inside will have to be manually put there, such as 'a' tags. The class and id of the tag can also be specified for css usage. A dropdown provides access to the content of whatever is stored inside the tag.

A note about image tags, they're specified in the same way, and the content of the tag on the input page, is used as the source of the image tag itself.

The meta

The meta tags allow you to add information to the page that is not displayed by default. This can be used to add extra information like date of creation, author, a short description of the page. All meta attributes are exposed to an api where you can request them using javascript.
